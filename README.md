# Online Order Management System for Bakeries 

A full-stack web application for an online bakery management and ordering system built with Python, HTML, JavaScript, and CSS.

## 📋 Overview

Online Order Management System for Bakeries is a comprehensive platform designed to streamline bakery operations and provide customers with an intuitive interface for browsing, ordering, and managing bakery products online.

## 🛠️ Tech Stack

- **Backend:** Python with Django
- **Frontend:** HTML, JavaScript, Tailwind CSS
- **Database:** PostgreSQL / SQLite
- **Admin Panel:** Django Unfold

## ✨ Features

### Customer Features
- Product catalog and browsing
- Online ordering system
- User account management
- Order tracking
- Responsive web design
- Shopping cart functionality

### Admin Features
- **User-friendly Admin Interface:** Django Unfold Admin Panel simplifies navigation for admins
- **Dashboard Analytics:** Provides insights into application usage, sales performance, and order metrics
- **Content Management:** Manage products, users, and orders from a centralized dashboard

## 🚀 Getting Started

### Prerequisites

- Python 3.x
- pip (Python package manager)
- Modern web browser

### Installation

1. Clone the repository:
```bash
git clone https://github.com/aarchi-palikhel/Online-Order-Management-System-for-Bakeries.git
cd Live_Bakery
```

2. Install dependencies:
```bash
pip install -r requirements.txt
```

3. Configure Django settings for Django Unfold:
   - Add Django Unfold to your `INSTALLED_APPS` in `settings.py`
   - Configure the necessary middleware and admin settings

4. Run the application:
```bash
python manage.py runserver
```
